Конфигурация сети состоит из четырех основных элементов:
1. IP адрес
2. сетевая маска
3. DNS-серверы
4. шлюз или маршутизатор, используемый по умолчанию для данного компа

Output net interfaces: '$ip addr'
Linux IP routing table: '$netstat -nr'

NS Record
NS record (nameserver record) is a DNS record that contains the name 
of the authoritative name server within a domaing or DNS zone.
